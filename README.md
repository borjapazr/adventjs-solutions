## 🚀 Quick start

- Install

  ```bash
  make install
  # or
  npm install
  ```

- Run all challenges

  ```bash
  make run/all
  # or
  npm test
  ```

- Run specific challenge

  ```bash
  make run/challenge-XX # where XX is the challenge number
  # or
  npm test XX # where XX is the challenge number
  ```
